"Takが作ったTODO List"

<追加点>
・TODOを完了しなければならない日(締切日のようなもの、期限)を記入できるようにして、
  内容と一緒に表示しました。
・期限の入力は、-(ハイフン)なしで「西暦+月+日」を入力します(例:2016年07月07日ならば 20160707)  。表示は、「西暦-月-日」。
・「新しいTODOを記入してください」という入力フォームに入力してEnterキーを押すと、
   追加ボタンを押したときと同じ動作を行います(画面に表示とデータの保存)。
・表示(追加)したTODOを全部クリアできるボタンを追加しました。押したときに出るアラートでは、全部クリア  のキャンセルができます。

<プログラムについて>
・仕様から、「投稿データはローカルストレージに保存する。」では、オブジェクトとして「期限」と「TODO」のデータをローカルストレージに保存しました。そこでは、オブジェクトをJSON文字列に変換→戻 して利用しました。
・仕様から、「新しい投稿ほど上に表示される。」では、keyを昇順にソートしてからkeyに対応する  valueを表示しました。
・普段使用しているC++では構造体をしばしば使いますが、JavaScriptにはその概念がないということで、  それを関数を用いて実現しました。

<その他>
・アラートはSweetAlertプラグインを使用しました。
